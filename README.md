# 🚗 AutoDeals.pk Clone | Full-Responsive Web Architecture

A high-fidelity frontend clone of **AutoDeals.pk**, Pakistan's leading automotive marketplace. This project demonstrates advanced layout structuring, responsive design principles, and the seamless integration of **Bootstrap 5** with **Custom CSS** to achieve a pixel-perfect commercial UI.

[**🌐 Experience the Live Demo**](https://autodeals-pk-app-clone-by-zestymec.netlify.app/)

---

## 🚀 The Challenge

The goal was to replicate a complex, data-heavy e-commerce platform. Automotive sites require precise navigation, high-quality image grids, and a multi-layered filtering system. I focused on maintaining the brand's aesthetic while ensuring the site remains lightning-fast and mobile-responsive.

## ✨ Core Features & UI Components

* **Complex Navigation:** A multi-level Mega Menu built for deep site navigation.
* **Responsive Car Grids:** Dynamic card layouts that adapt perfectly to Mobile, Tablet, and Desktop views.
* **Search & Filter UI:** A detailed search interface replicating the real-world automotive search experience.
* **Custom Styling:** Overrode Bootstrap defaults with **Pure CSS** for brand-specific typography and spacing.
* **Optimized Assets:** Managed high-resolution car imagery without sacrificing page load performance.

## 🛠️ Tech Stack & Skills Demonstrated

* **Frontend:** HTML5, CSS3 (Advanced Flexbox & Grid)
* **Framework:** Bootstrap 5 (Components & Grid System)
* **Responsiveness:** Media Queries & Fluid Typography
* **Tooling:** Git/GitHub for Version Control
* **Hosting:** Netlify (Continuous Deployment)

## 📂 Project Architecture

Recruiters can find the core logic organized as follows:
* `index.html`: Semantically structured document for SEO friendliness.
* `css/style.css`: Modular custom CSS overriding framework defaults.
* `images/`: Compressed and optimized assets for fast LCP (Largest Contentful Paint).

## 🧠 Engineering Insights

During this build, I prioritized **Semantic HTML** to ensure the site is accessible (A11y). One of the biggest hurdles was managing the complex Bootstrap navbar with custom hover effects, which I resolved by utilizing CSS pseudo-classes to maintain a "Human-centric" user experience.
